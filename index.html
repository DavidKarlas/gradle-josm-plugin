Gradle plugin for JOSM (plugin) development

Groovydoc of the plugin

There is a Maven repository containing the plugin at https://floscher.github.io/gradle-josm-plugin/maven, you can view its contents at https://github.com/floscher/gradle-josm-plugin/tree/gh-pages/maven.

Add the following lines to your build.gradle to start using the plugin: 

buildscript {
  repositories {
    maven {
      url 'https://floscher.github.io/gradle-josm-plugin/maven'
    }
  }
  dependencies {
    classpath group: 'org.openstreetmap.josm.gradle.plugin', name: 'josm-plugin', version: '0.1.0'
  }
}
