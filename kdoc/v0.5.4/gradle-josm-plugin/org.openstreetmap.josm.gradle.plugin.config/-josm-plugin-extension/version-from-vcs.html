versionFromVcs

var versionFromVcs: Boolean
Deprecated: Will soon be removed with release 0.6.0

Determines if the version number of the project should be derived from the version control system (git, or SVN if git is not present).

In case neither git nor SVN is present, the project version will not be modified regardless of this setting.

Default value: true

Since
0.5.2
