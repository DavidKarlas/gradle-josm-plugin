logJacocoCoverage

var logJacocoCoverage: Boolean

Iff this is true, the coverage numbers for instruction, branch and line coverage are logged for every tasks with type org.gradle.testing.jacoco.tasks.JacocoReport.

The format is as follows:

Instruction coverage: 25.0000 % (25 of 100)
     Branch coverage: 50.0000 % (5 of 10)
       Line coverage: 75.0000 % (3 of 4)

The message is printed via Project.getLogger, logging level is org.gradle.api.logging.Logger.lifecycle.

Default value: true
