iconBase64Provider

@Internal var iconBase64Provider: (String) -> String?

Optional parameter, converts a relative icon path (you decide relative to what, this class does not make assumptions about that) to a Base64 representation. This parameter is optional, by default or if it returns null, the icon path is added as-is to the list.
