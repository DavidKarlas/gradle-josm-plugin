AddMinJosmVersionDependency()

Add the dependencies of the implementation configuration of one source set (mainSourceSet) to the implementation configuration of another source set (minJosmVersionSourceSet).

The version of the JOSM dependency is set to JosmManifest.minJosmVersion. When this task runs, it is checked, if that version is available, if not a later version is chosen according to getNextJosmVersion.
