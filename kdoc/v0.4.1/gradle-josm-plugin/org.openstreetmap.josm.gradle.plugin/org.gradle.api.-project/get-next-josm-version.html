getNextJosmVersion

fun Project.getNextJosmVersion(startVersion: String?): Dependency

Finds the next JOSM version available in the repositories known to Gradle, starting with a given version number.

If the number can be parsed as an integer, 50 version numbers beginning with the given number are checked. Otherwise only the given version is checked.

Exceptions

GradleException - if no suitable JOSM version can be found.

Return
the next available JOSM dependency. A DependencySet consisting only of this one dependency.
